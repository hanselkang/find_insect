# Insects Finder
  
</div>


## Features
----------

* Find Insects and their informations by
  - Distribution
  - UK Status
  - Crops
  - Continents
* Search Insects By Name 
* View selected insect detail
* Sort Insects b
  - Title
  - ID



## Technology
----------

* Node.js
* React.js
* CSS
* OpenAPI https://insectvectors.science/api/vectors
